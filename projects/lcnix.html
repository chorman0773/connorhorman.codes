What is LCNix

LCNix is a project to produce a Linux Standards Base compliant operating system, replacing both the linux kernel and GNU Software with custom software. LCNix also will provide a number of replacements for standard GNU Tools, including the GNU Compiler Collection, Coreutils, and even binutils with LC Software. The LCNix Kernel, written primarily in rust, along with some C and assembly for the x86_64 architecture, will be a POSIX compliant OS Kernel, and provide the same userspace system call ABI as the linux kernel
