Laser

What is Laser

Laser is a Work In Progress Systems Programming Language with a focus on Game Development. The language has some roots in C and Rust. It is developed by a committee formed from 6 working groups, with various focuses. I chair the Embedded Domain Working Group and run the mailing list for that working group.
